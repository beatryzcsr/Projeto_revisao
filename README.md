Descrição
==========
Desenvolvemos uma aplicação web completa (Full Stack) para gerenciar o vínculo entre Empresas (Fornecedoras) e seus Produtos. A arquitetura consiste em um backend em Node.js estruturado como uma API RESTful e um frontend SPA (Single Page Application) construído em React.
